### Upgrade Guide

> Do **NOT** delete `resources folder`, `secrets.env` or `settings.json`

  * Stop the server

  * Delete all files & folders in the application root folder except `resources folder`, `secrets.env` and `settings.json`

  * Copy new files

  * Run `npm install`

  * Start the server
